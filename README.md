# The Slap Blog ✨

Um blog interativo e diário de emoções inspirado no The Slap.  
Aqui você pode registrar seus sentimentos, postar pequenas reflexões e se divertir usando emojis para mostrar seu humor do dia! 😛  

---

## 🎯 Funcionalidades
- Posts interativos com botão **“Brilhar ✨”**  
- Diário de emoções com emojis fáceis de clicar  
- Histórico de entradas com data para acompanhar seus sentimentos  
- Layout moderno e colorido, direto no navegador  

---

## 🚀 Como usar
1. Abra o arquivo `index.html` no navegador ou no Acode.  
2. Escreva como você se sente no diário e clique em **Salvar**.  
3. Use os botões de emoji para adicionar sentimentos ao texto.  
4. Clique nos posts no botão **“Brilhar ✨”** para interagir com eles.  

---

## 🛠 Tecnologias usadas
- HTML  
- CSS  
- JavaScript  

---

## 💡 Observações
- Tudo funciona direto no navegador, não precisa instalar nada.  
- As entradas do diário ficam salvas localmente no navegador (localStorage).  
- Divirta-se e registre seus momentos diários! 😎
